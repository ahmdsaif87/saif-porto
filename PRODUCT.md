# Product

## Register

brand

## Users

Recruiters, hiring managers, freelance clients, and fellow developers visiting to evaluate skills, explore project work, and decide whether to hire or collaborate.

## Product Purpose

A personal portfolio showcasing Ahmad Saif's work as a software engineer. The portfolio itself is the design statement — minimal, confident, and technically excellent. Every interaction should communicate craft and attention to detail.

## Brand Personality

Minimal, clean, confident. No-nonsense. Restrained typography. Brutalist-adjacent in its honesty. Speaks through substance, not decoration.

## Anti-references

- No AI-generated UI tells: purple gradients, glassmorphism, bounce/elastic animations, side-stripe accent borders, gradient text
- No SaaS landing page clichés: hero-metric sections, testimonial carousels, feature-grid-with-icons
- No generic dev-portfolio tropes: terminal themes, glowing neon, matrix aesthetics
- No corporate template feel: no stock-photo vibes, no generic business-card layouts

## Design Principles

- **Show, don't tell** — the interface itself demonstrates skill. Every pixel, interaction, and animation is a portfolio piece.
- **Confident restraint** — say more with less. Generous whitespace, deliberate typography, judicious use of color and motion.
- **Brutalist honesty** — no decorative cruft. Borders are real borders (radius: 0). Shadows don't exist. Structure is revealed through layout, not ornament.
- **Substance over slop** — reject every AI-generated reflex. If it looks like it could have been templated, it doesn't belong.

## Accessibility & Inclusion

- WCAG AA minimum (4.5:1 body text contrast, 3:1 large text)
- Proper focus states and keyboard navigation
- Reduced motion support via `prefers-reduced-motion`
- Semantic HTML throughout
